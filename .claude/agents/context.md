## Frontend Architecture & Current Design State (Phase 2 deep-dive)

### App shell & flow
- Single-page client app. `app/page.tsx` (`"use client"`) holds ALL state and a simple phase state machine: `phase: "gate" | "live"`, plus a `conn` union (`idle | requesting | incoming | connecting | connected`) and a `video` union (`none | requesting | incoming | active`).
- **User flow**: Landing (`EntryGate`) → click "Enter Pulse" → browser geolocation prompt → on success calls `join()` and flips to `live` → full-screen `WorldMap` with peer dots → tap a dot → "Requesting connection…" toast → peer gets `ConnectionPrompt` ("A stranger wants to connect") → on accept, WebRTC connects → `ChatPanel` slides in (right drawer) → optionally tap "Video" → `ConnectionPrompt` ("Start video call?") on peer → `VideoPanel` takes over full screen. "End" / "End video" tears down.

### UI surfaces (every screen/overlay)
1. **EntryGate** (`app/components/EntryGate.tsx`) — Landing screen. Centered column on `bg-zinc-950`: `Pulse` title (`text-4xl font-bold`), tagline, single emerald CTA button ("Enter Pulse" / "Locating…" disabled state), error text (`text-red-400`), privacy footnote. Status states: `idle | locating | error`. No spinner/animation currently — just a label swap.
2. **WorldMap** (`app/components/WorldMap.tsx`) — Full-screen `dark-v11` Mapbox map. Peer dots = `.pulse-dot` (14px, white ring, per-id HSL color via hash, `pulse-ring` keyframe animation, hover scale 1.3, opacity 0.35 when busy). Own location = `.pulse-me` emoji 📍 pin with a "Me" label. Bottom-left floating pill shows `{n} online`. Fallback message if no Mapbox token.
3. **ConnectionPrompt** (`app/components/ConnectionPrompt.tsx`) — Reusable centered modal over `bg-black/60`. Card: `rounded-2xl bg-zinc-900`, title + optional subtitle, two buttons (outline Decline + emerald Accept). Used for both incoming connection and incoming video requests.
4. **ChatPanel** (`app/components/ChatPanel.tsx`) — Right-anchored full-height drawer (`max-w-md`, `border-l border-zinc-800 bg-zinc-950`). Header: "Stranger" + Connected/Connecting status, Video + End buttons. Scrollable message list (own = emerald bubble right-aligned, peer = zinc bubble left-aligned, `rounded-2xl`), auto-scroll to bottom. Bottom input row (rounded input + emerald Send), disabled until connected.
5. **VideoPanel** (`app/components/VideoPanel.tsx`) — Full-screen `bg-black`. Remote video fills screen (`object-cover`), local video as bottom-right PiP (`h-40 w-28 rounded-lg`). "Waiting for stranger's video…" placeholder. Bottom bar with red "End video" button.
6. **Floating toasts/notices** (inline in `page.tsx`) — Top-center pills (`bg-zinc-800/90 backdrop-blur rounded-full`): transient `notice` messages, "Requesting connection…" (+Cancel), "Waiting for stranger to accept video…".

### Current visual / design system (as-built)
- **Color palette**: Dark, near-monochrome. Surfaces `zinc-950 / zinc-900 / zinc-800`; text `zinc-100 / zinc-400 / zinc-500 / zinc-600`; borders `zinc-700 / zinc-800`. Single accent: `emerald-400` (CTAs, own chat bubbles, links, focus ring), hover `emerald-300`. Destructive: `red-500` (End). Errors: `red-400`. Peer dots use generated per-id HSL hues (only place with varied color).
- **Typography**: Geist Sans / Geist Mono loaded via next/font but body currently falls back to Arial (globals.css override — flag for Phase 2). Headings `text-4xl font-bold` (gate), `text-lg font-semibold` (modal), `font-semibold` (panel headers). Body `text-sm`, hints `text-xs`.
- **Spacing/shape**: Generous gaps (`gap-3/5/8`), padding `p-4/p-6`. Heavy use of `rounded-full` (buttons, pills, input) and `rounded-2xl` (cards, bubbles). Backdrop blur on floating overlays. Shadows `shadow-lg / shadow-xl / shadow-2xl`.
- **Motion**: Minimal. Only `pulse-ring` keyframe on dots + Tailwind `transition` hover states. No enter/exit transitions, no layout animation, no skeleton/loaders.
- **Design tokens**: Only `--background`, `--foreground` (+ dark-mode media query) and font vars in the `@theme` block of `globals.css`. No spacing/radius/color token scale beyond Tailwind defaults. No documented design system.
- **Screens a redesign would touch**: all six surfaces above — primarily EntryGate (first impression), WorldMap (the hero/core experience), ChatPanel + VideoPanel (interaction surfaces), ConnectionPrompt + toasts (micro-interactions). Globals.css is the place to introduce a real token/theme layer.

## Constraints
